# capybartoon.github.io
CapyBartoon es un universo creativo protagonizado por una capibara que combina humor absurdo, animación 3D e ideas sin límites. Un proyecto innovador de entretenimiento visual que, además de alegrar con cortosmetrajes originales, ofrece modelos 3D de capibaras con una amplia variedad de estilos para creadores y artistas curiosos.
